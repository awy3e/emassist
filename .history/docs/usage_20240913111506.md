
# Usage Guide

## Installation

Follow these steps to install Emassist:

1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/emassist.git
    ```

2. Build Emassist:
    ```bash
    cd emassist
    ./bin/emassist build
    ```

## Basic Commands

- `emassist run` - Start Emassist.
- `emassist test` - Run tests.
